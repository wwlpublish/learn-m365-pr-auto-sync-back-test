Using Adaptive Card Extensions, you can extend Viva Connections on the mobile and desktop experience. Adaptive Card Extensions allow you to provide editors with configurable cards that they can place on the dashboard that power Viva Connections.

In this module, you learned to:

- Install development tools for building Microsoft Viva Connections extensions.
- Extend Microsoft Viva Connections with Adaptive Card Extensions.
- Deploy and test custom Adaptive Card Extensions in Microsoft Viva Connections.
