In this module you have learned about DLP policies, what they are, how they work, and how they are used in Microsoft Teams. You have learned how to gauge the effectiveness of policies by using reports and writing queries that can be used with e-Discovery.

Now that you have completed this module, you will be able to:

- Describe what a DLP policy contains.
- Describe how DLP policies work.
- Understand how DLP and Microsoft Teams work.
- Create a DLP policy from a template.
- Create a custom DLP policy.
- Customize policy tips.
- Prevent external access.
- Use reports to gauge effectiveness of your policies.
