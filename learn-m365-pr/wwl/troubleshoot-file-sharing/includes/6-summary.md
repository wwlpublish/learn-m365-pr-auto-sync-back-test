In this module you have learned how to troubleshoot file sharing issues in Teams and how to edit file sharing settings. You have learned about file sharing in public and private channels, including where files are stored.

Now that you have completed this module, you are able to:

- Troubleshoot private chat (P2P) file issues.
- Troubleshoot file issues for private channels.
- Troubleshoot file issues for public channels.
