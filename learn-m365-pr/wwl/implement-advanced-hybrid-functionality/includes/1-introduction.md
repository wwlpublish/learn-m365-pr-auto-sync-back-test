Once an organization implements a hybrid deployment to connect its on-premises Exchange environment to Exchange Online, it must then manage the deployment. Managing a hybrid deployment enables you to implement advanced functionality that wouldn't be possible without a hybrid deployment.

In this module, you'll learn how to manage a hybrid deployment and implement advanced hybrid functionality. You'll also examine the features that require a successful hybrid deployment, such as Public Folder coexistence and OneDrive for Business attachment storage for on-premises mailboxes.

### Prerequisites

This module is designed for persons who are aspiring to the Microsoft 365 Messaging Administrator role. The prerequisites for this module include:

 -  Ability to navigate the Microsoft 365 admin center, the Exchange admin center, and the Microsoft 365 Defender portal.
 -  Ability to create Domain Name System (DNS) records at an intermediate level.
 -  Familiarity with Active Directory concepts such as centralized domain management, sites, and directory-based, identity-related services.
 -  Ability to write PowerShell commands at an intermediate level.

### Learning objectives

After completing this module, you'll be able to:

 -  Manage a hybrid deployment.
 -  Describe when you must configure Public Folder coexistence with Microsoft 365.
 -  Explain how to configure Oauth for a mixed Exchange environment.
 -  Describe how to configure OneDrive attachments for on-premises mailboxes.
